## Example output
**Title:** Enterprise Asset-Tagging Engine  
**Status:** Approved  
**Stakeholders:** Sarah Jenkins (Product), Mark Wu (Engineering)

**Problem Statement:**
Enterprise users currently spend 12+ hours per week manually tagging cloud assets across many regions. This causes cost-allocation errors and compliance gaps.

**User Stories:**
- As a DevOps Lead, I want an AI-assisted tagging suggestor so that I can keep production resources compliant.
- As a Finance Manager, I want tags validated against the active cost-center database so orphan billing is prevented.

**Functional Requirements:**
1. Bulk tagging interface supporting large views.
2. Schema validator for tag rules.
3. AI suggestion engine based on asset metadata and history.

**Success Metrics:**
- 50% reduction in untagged assets within 30 days.
- <1% cost-center validation error rate.
