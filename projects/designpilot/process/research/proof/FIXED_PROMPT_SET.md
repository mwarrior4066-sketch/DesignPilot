# Fixed Prompt Set

## Prompt 01 — baseline comparison
Rebuild the explanation for a design-ops AI tool so it communicates the problem, process, solution, and outcome clearly without pretending there is validation where none exists.

## Prompt 02 — structural critique
Audit a cluttered dashboard first view and decide what should move, what should stay, and what question the dashboard should answer first.

## Prompt 03 — remediation realism
Write a PDF remediation plan that preserves semantic structure, reading order, extraction quality, and verification discipline instead of only proposing visual cleanup.
