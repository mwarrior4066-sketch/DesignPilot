# User Journey Map

## Required fields
- Stage
- User goal
- Action
- Decision friction
- Touchpoints
- Failure risk
- Opportunity
- Evidence source

## Field standards
### Decision friction
- Good looks like: names the exact moment the user hesitates, confuses states, or loses trust
- Minimum detail: 2-3 sentences tied to a real stage transition
- Failure mode: emotional language with no action consequence

### Opportunity
- state the intervention and why it belongs at this stage instead of later
- include what constraint the intervention must preserve

## Anti-patterns
- generic empathy map phrasing
- opportunities that are only “make it simpler”
- no evidence source or observed trigger
