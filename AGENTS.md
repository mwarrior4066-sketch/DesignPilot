# DesignPilot Agent Map

## Where to start
- operator deployment artifacts: `dist/`
- lightweight deploy surface: `dist/DEPLOY_LITE.md`, `dist/lite_routes/`, `dist/lite_contracts/`, `dist/lite_starters/`
- operator docs: `docs/operator/`
- maintainer docs: `docs/maintainer/`
- modular source of truth: `src/`
- evaluations: `evals/`
- public proof: `proof/`

Treat `dist/` as the deployment surface and `src/` as the source of truth.
