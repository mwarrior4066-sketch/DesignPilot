# Grid Mediums Summary

Primary sources: Grid Systems Guide and AI PDF Creation Grid and Typography.

## Print and PDF
- use a manuscript grid for long uninterrupted reading
- use a two-column or asymmetrical editorial grid for mixed reading plus support content
- use a 12-column modular grid for mixed media, tables, charts, and flexible page systems
- preserve baseline rhythm and independent text-frame logic in fixed layouts

## Web
- default to a responsive 12-column grid
- contract logically to 8 columns on tablet and 4 or 1 on mobile
- use CSS Grid when rows and columns both matter

## App UI
- use an 8pt spacing system
- default to a 4-column mobile layout
- respect touch targets and safe areas

## Slides
- default to 16:9 with a 12-column grid
- use 6 or 8 columns only when that makes the deck clearer
- pair the structural grid with a focal composition rule such as rule-of-thirds placement
