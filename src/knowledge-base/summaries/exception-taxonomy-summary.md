# Exception Taxonomy Summary

## Purpose
Quick summary of the major operator failure classes: phase, source-of-truth, accessibility, grid inference, dashboard density, component drift, PDF integrity, token architecture, implementation certainty, routing ambiguity, and iteration loops.

## Use when
- the answer needs a safe fallback
- two pathways compete
- the user is requesting a known failure state

## Default move
- classify the failure
- choose the smallest safe fallback
- avoid fake certainty
