# Research Prompt Pack Template

## Rule
Use only as many prompts as needed, up to 4 maximum.
Each prompt must be materially distinct, not a cosmetic rewrite.

## Pack metadata
- Prompt pack name:
- Related project:
- Related decision:
- Version:
- Last updated:

## Research goal
- state the decision this research will unblock

## What is already known
- list the facts, constraints, or artifacts already in hand

## What is missing
- name the proof, comparison, or implementation gap still open

## Scope boundary
- name what the research should not drift into

## Evidence target
- define the proof shape needed: benchmark, policy, market signal, workflow evidence, case-study proof, or implementation rule

## Exclusions
- name the kinds of sources or advice that should be avoided

## Output format
- say exactly what the research model should return

## Anti-patterns
- four prompts that ask the same thing four ways
- prompts with no output format
- generic “find best practices” wording
- prompts that do not say what the evidence should prove
