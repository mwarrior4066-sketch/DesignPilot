## Example output
**Screen:** Global Resource Monitor

**Primary Goal:**
Provide an at-a-glance view of all cloud resources and their immediate health status.

**Hierarchy:**
1. Critical: total active alerts and high-priority cost anomalies
2. Primary: resource list with health indicators
3. Secondary: metadata sidebar

**Components:**
- status badge
- global search
- filter chips
- alert summary cards

**Interactions:**
- search with autocomplete
- filter by region/type/cost center
- expand a resource for metadata details
