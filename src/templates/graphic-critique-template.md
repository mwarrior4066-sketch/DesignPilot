## Example output
### Format Classification
- Poster / campaign / deck / editorial / system graphic: keynote slide for a live strategy presentation
- Viewing time class: short, roughly 5–12 seconds before narration moves on
- Viewing distance assumption: room-scale viewing from front row to back wall

### Communication Contract
- Primary message: “Compliance exceptions are rising fastest in two regions, not across the board”
- Secondary message: supporting chart that shows the two-region spike
- Tertiary information: source timestamp and data scope note

### Hierarchy Audit
- Dominant focal node: the headline should be dominant, but the oversized multicolor chart title is competing with it
- Rival focal nodes: chart labels and a decorative callout badge are too close in weight and saturation to the main statement
- Hierarchy depth: effectively 5 levels on one slide, which is too deep for this viewing window

### Composition Audit
- Type-led / image-led / dual-led: intended dual-led, but currently acting like three competing focal fields
- Grid logic: basic two-field layout is present but not reinforced by spacing or alignment
- Intentional breaks: decorative badge breaks the grid without earning attention value

### Legibility Audit
- Distance/type concerns: supporting annotation text is too small for back-row viewing and should be promoted or removed
- Contrast concerns: chart category colors are vivid but the grey supporting note is too low-contrast against the projected background
- Noise / clutter concerns: too many category colors and repeated axis labels create unnecessary edge activity
- Attention-window mismatch: the slide asks for detailed reading when the format only supports rapid comprehension

### Fix Plan
1. Collapse the message into one dominant headline and move the chart title into a supporting caption role.
2. Reduce category count or isolate the two critical regions so the chart reinforces the claim instead of competing with it.
3. Remove the decorative badge and use spacing plus one accent color to restore a clean hierarchy.
