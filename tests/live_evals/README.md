### What this directory is for
Live and proxy comparative runs between pack mode (system prompt active) and baseline mode (no system prompt).
The `live-run-*` directories under `tests/live_outputs/` hold single-model pass/fail evidence;
this directory is for side-by-side comparative evidence.
