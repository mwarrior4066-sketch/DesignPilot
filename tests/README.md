# Tests

This directory contains regression fixtures, golden outputs, eval artifacts, scorecards, and live-eval scaffolding.

## Layers
- `fixtures/` prompt inputs
- `golden_outputs/` known pass/fail outputs
- `evals/` stored evaluation results
- `scorecards/` rubric definitions
- `live_evals/` blinded comparative run storage

## v2.5.0 additions
- live-eval protocol and schema
- hollow-compliance and visual-boundary fail cases
- validator coverage for degraded mode and lightweight execution
