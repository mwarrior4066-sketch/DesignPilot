## Architectural framing
The system needs a source-of-truth model and an authority posture.

## Authority and consistency
Use actor, resource, and action rules because they govern access. Use a source-of-truth boundary because it governs data views. Use a consistency model because it governs shared state.

## Delivery notes
Webhooks, exports, and pagination each need explicit system rules.
