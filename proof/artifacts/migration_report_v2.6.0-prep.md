# Migration Report - v2.6.0-prep

## Structural changes
- introduced `src/` as the modular source layer
- introduced `config/`, `docs/`, `evals/`, and `proof/`
- added a first source-to-artifact compiler
- generated operator-facing deploy files in `dist/`

This is a first implementation pass. It establishes the architecture needed for productization without claiming that the external proof layer is already complete.
