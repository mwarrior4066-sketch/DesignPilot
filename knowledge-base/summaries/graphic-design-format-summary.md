# Graphic Design Format Summary

## Purpose
Canonical summary for graphic-communication formats where the key question is what must be perceived, by whom, at what distance or time, and under what hierarchy constraints.

## Use when
- the task is a poster, campaign asset, editorial spread, slide deck, key visual, or composition-heavy graphic system

## Default rules
- classify the format before styling it
- define viewing-time class and distance class before density decisions
- define one dominant focal node per view unless a deliberate dual-focal system is justified
- set hierarchy depth by format, not by personal preference
- campaign and distance-driven work require brevity and stronger type-size/contrast discipline than editorial work
- deck logic is not magazine logic; deck logic is time-based and distance-based
- long-form deck/report work should distinguish cover, TOC, chapter divider, content, and close roles when wayfinding matters
- section markers should work as landmarks, not decorative noise
- use temperature shifts instead of harsh brightness inversions when a dark deck needs rhythm
- dynamic content boxes and vertical-fill math should prevent floating cards and large bottom dead zones in slide systems
- one-idea-per-slide is the safer default for live presentation contexts; split overloaded pages before shrinking type

## Format contracts
- Campaign / OOH: ultra-short attention window, one primary message, one brand, one action
- Poster: stronger hierarchy, some dwell time, but still distance-aware and sparse
- Editorial: sustained reading, rhythm, line length, baseline, spread continuity
- Presentation / Deck: shared viewing at distance, sequential comprehension, reduced density
- System-led brand graphic: repeatable rules, controlled variance, consistent assets

## Useful thresholds
- body reading measure stays in the readability-safe range for editorial work
- distance-driven work needs larger type and stronger contrast than near-reading work
- all-caps increase legibility burden in distance contexts
- if more than one element reads as the headline, hierarchy is unstable
- if content volume exceeds the format’s attention window, the format is wrong or the content must split

## Failure patterns
- deck treated like editorial
- billboard treated like brochure
- multiple rival focal points
- thin text over busy imagery
- decorative texture replacing information hierarchy
- grid-breaking that creates chaos instead of emphasis
