# UX Cognition Summary

Primary sources: Comprehensive UI/UX Project Roadmap and Design Strategy and Communication Research.

## Key topics
- cognitive load
- mental models
- Laws of UX
- fixation and saccade behavior
- thumb zones and anthropometry
- visual scanning patterns

## Operational rules
- reduce extraneous cognitive load
- chunk information instead of overloading memory
- respect familiar interaction patterns unless a better pattern is justified
- place high-frequency mobile actions in natural thumb reach
- design for scanning, not fantasy full reading
