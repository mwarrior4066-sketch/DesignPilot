# Graphic Design Format Summary Runtime Summary

## Decision rules
- classify the format before styling it
- define viewing-time class and distance class before density decisions
- define one dominant focal node per view unless a deliberate dual-focal system is justified
- set hierarchy depth by format, not by personal preference
- campaign and distance-driven work require brevity and stronger type-size/contrast discipline than editorial work
- deck logic is not magazine logic; deck logic is time-based and distance-based
- long-form deck/report work should distinguish cover, TOC, chapter divider, content, and close roles when wayfinding matters
- section markers should work as landmarks, not decorative noise
- use temperature shifts instead of harsh brightness inversions when a dark deck needs rhythm
- dynamic content boxes and vertical-fill math should prevent floating cards and large bottom dead zones in slide systems
- one-idea-per-slide is the safer default for live presentation contexts; split overloaded pages before shrinking type

## Failure traps
- deck treated like editorial
- billboard treated like brochure
- multiple rival focal points
- thin text over busy imagery
- decorative texture replacing information hierarchy
- grid-breaking that creates chaos instead of emphasis
